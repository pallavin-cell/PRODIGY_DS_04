# PRODIGY_DS_04

## Task 04 - Sentiment Analysis on Social Media Data

### Objective
Analyze and visualize sentiment patterns in social media data to understand public opinion and attitudes toward specific topics or brands.

### Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

### Workflow
- Data Loading
- Data Cleaning
- Exploratory Data Analysis
- Text Vectorization using CountVectorizer
- Model Training using Multinomial Naive Bayes
- Model Evaluation
- Sentiment Prediction

### Results
- Visualized sentiment distribution.
- Achieved good classification accuracy.
- Generated a confusion matrix and classification report.
- Predicted sentiments for custom input sentences.
